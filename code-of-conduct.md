# Code of Conduct

## Short version

Be respectful. Disagree on tools, not on people. We're all here because we like good terminal software.

## Longer version

This project and everyone participating in it is expected to treat each other with respect. This applies to issues, pull requests, discussions, and any other interaction within the project.

Specifically:

- Be welcoming to newcomers
- Respect differing opinions on tools and technologies
- Accept constructive criticism gracefully
- Focus on what's best for the list and its users

Harassment, trolling, and personal attacks are not tolerated.

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any other channel related to this project.

## Enforcement

Instances of unacceptable behavior can be reported by opening an issue or contacting the maintainer directly. All reports will be reviewed.

Project maintainers who do not follow or enforce the Code of Conduct may be removed from the project team.
